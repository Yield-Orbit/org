链接设置

所有链接配置

以下是所有产品的链接设置区域，请将您的实际链接填入对应的位置

统计小程序链接

要设置统计小程序的链接，请找到id为"statAppLink"的a标签，在href属性中填入您的小程序链接地址。

代码位置：

<a href="#" class="product-link app-link" id="statAppLink">

替换为：

<a href="https://your-stat-app-link.com" class="product-link app-link" id="statAppLink">

养成游戏链接

要设置养成游戏的链接，请找到id为"gameLink"的a标签，在href属性中填入您的游戏链接地址。

代码位置：

<a href="#" class="product-link game-link" id="gameLink">

替换为：

<a href="https://your-game-link.com" class="product-link game-link" id="gameLink">

论坛链接

要设置论坛的链接，请找到以下两个id的a标签，在href属性中填入对应的论坛链接地址。

小程序论坛：

<a href="#" class="product-link forum-link" id="appForumLink">

替换为：

<a href="https://your-app-forum.com" class="product-link forum-link" id="appForumLink">

游戏论坛：

<a href="#" class="product-link forum-link" id="gameForumLink">

替换为：

<a href="https://your-game-forum.com" class="product-link forum-link" id="gameForumLink">
